# Changelog

## 1.0.0 — 2026-07-05

Initial release.

- Synced lyrics panel for YouTube music videos (LRCLIB primary, NetEase fallback)
- Multi-strategy song matching with duration corroboration and
  cross-provider canonical resolution
- Automatic romanization for Devanagari, Gurmukhi, Bengali, Tamil, Telugu,
  Kannada, Malayalam, Korean, and Japanese kana
  (Original / Romanized / Both display modes)
- Slowed + reverb / sped-up support via timestamp rescaling;
  live performances shown as untimed lyrics
- Glassmorphic docked side panel with slide animation, resizable width,
  thin collapsed tab, Alt+L toggle, per-session lyric caching
